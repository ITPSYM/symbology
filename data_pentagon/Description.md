# Pentagon alphabet (n = 31)

## Independent variables

The alphabet is built on **5 independent Mandelstam invariants**:

```
{s12, s23, s34, s45, s15}
```

plus a single square-root variable `eps5`, defined by `RootDef` in
[alphabet.wl](alphabet.wl):

```
eps5 = Sqrt[Delta5]
```

where `Delta5` is the 5-point Gram determinant (a quartic polynomial in the
five `s`'s):

```
Delta5 = s12^2 s15^2 - 2 s12^2 s15 s23 + s12^2 s23^2
       + 2 s12 s15 s23 s34 - 2 s12 s23^2 s34 + s23^2 s34^2
       - 2 s12 s15^2 s45 + 2 s12 s15 s23 s45 + 2 s12 s15 s34 s45
       + 2 s12 s23 s34 s45 + 2 s15 s23 s34 s45 - 2 s23 s34^2 s45
       + s15^2 s45^2 - 2 s15 s34 s45^2 + s34^2 s45^2
```

This is the unique algebraic letter for massless 5-particle kinematics.

## The 31 letters

### Rational letters — linear in the s's (W[1]–W[25])

Single invariants:

| | | | | |
|---|---|---|---|---|
| W[1] = s12 | W[2] = s23 | W[3] = s34 | W[4] = s45 | W[5] = s15 |

Sums of two adjacent:

| | | | | |
|---|---|---|---|---|
| W[6] = s34 + s45 | W[7] = s15 + s45 | W[8] = s12 + s15 | W[9] = s12 + s23 | W[10] = s23 + s34 |

Differences of two:

| | | | | |
|---|---|---|---|---|
| W[11] = s12 − s45 | W[12] = s23 − s15 | W[13] = s34 − s12 | W[14] = s45 − s23 | W[15] = s15 − s34 |

Linear combos of three:

| | | | | |
|---|---|---|---|---|
| W[16] = s12 + s23 − s45 | W[17] = s23 + s34 − s15 | W[18] = s34 + s45 − s12 | W[19] = s15 + s45 − s23 | W[20] = s12 + s15 − s34 |

Linear combos of four:

| | | | | |
|---|---|---|---|---|
| W[21] = s34 + s45 − s12 − s23 | W[22] = s15 + s45 − s23 − s34 | W[23] = s12 + s15 − s34 − s45 | W[24] = s12 + s23 − s15 − s45 | W[25] = s23 + s34 − s12 − s15 |

### Square-root letters (W[26]–W[31])

Each of W[26]–W[30] has the form `(P − eps5)/(P + eps5)` where `P` is a
bilinear in the `s`'s:

| Letter | P (so that letter = (P − eps5)/(P + eps5)) |
|--------|------|
| W[26] | −s12·s15 + s12·s23 − s23·s34 − s15·s45 + s34·s45 |
| W[27] | −s12·s15 − s12·s23 + s23·s34 + s15·s45 − s34·s45 |
| W[28] | +s12·s15 − s12·s23 − s23·s34 − s15·s45 + s34·s45 |
| W[29] | −s12·s15 + s12·s23 − s23·s34 + s15·s45 − s34·s45 |
| W[30] | +s12·s15 − s12·s23 + s23·s34 − s15·s45 − s34·s45 |

The five `P`'s are the five distinct sign flips (even number of flips each)
of the bilinear `s12·s15 − s12·s23 + s23·s34 + s15·s45 − s34·s45`,
reflecting the cyclic structure of the pentagon.

Finally:

```
W[31] = eps5^2 = Delta5
```

the polynomial under the square root itself. Although `W[31]` is rationally
expressible in the `s`'s, it is kept as a distinct letter because
`d log(eps5^2) = 2 d log(eps5)` ties it to the same square-root branch as
W[26]–W[30].

## Square-root handling

The pentagon alphabet is **not Galois-invariant**: the 5 algebraic letters
`W[26]-W[30]` of the form `(P - eps5)/(P + eps5)` are listed without their
conjugates `(P + eps5)/(P - eps5) = 1/W[i]`. Consequently the integrability
minors contain `1/Sqrt[Delta5]` factors that do **not** cancel.

[SymbolBootstrap.wl](../SymbolBootstrap.wl) handles this automatically:
`GenSqrtD` detects `Sqrt[Delta5]`, Z2-reduces the square-root set,
rationalizes denominators, and `GenIntRelMat` keeps `sq[i]` symbolic in the
minors, extracting rational coefficients via `CoefficientArrays`. All
equations stay in `QQ`, so SparseRREF works natively. No special handling
is required from the caller beyond substituting `eps5 -> Sqrt[Delta5]`
(via `RootDef`) in the alphabet expressions.
