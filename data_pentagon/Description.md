# Pentagon alphabet (n = 31)

## Independent variables

The alphabet is built on **5 independent Mandelstam invariants**:

```
{s12, s23, s34, s45, s15}
```

plus a single square-root variable `eps5`, defined by `RootDef` in
[alphabet.wl](alphabet.wl):

```
eps5 = Sqrt[Delta5]
```

where `Delta5` is the 5-point Gram determinant (a quartic polynomial in the
five `s`'s):

```
Delta5 = s12^2 s15^2 - 2 s12^2 s15 s23 + s12^2 s23^2
       + 2 s12 s15 s23 s34 - 2 s12 s23^2 s34 + s23^2 s34^2
       - 2 s12 s15^2 s45 + 2 s12 s15 s23 s45 + 2 s12 s15 s34 s45
       + 2 s12 s23 s34 s45 + 2 s15 s23 s34 s45 - 2 s23 s34^2 s45
       + s15^2 s45^2 - 2 s15 s34 s45^2 + s34^2 s45^2
```

This is the unique algebraic letter for massless 5-particle kinematics.

## The 31 letters

### Rational letters — linear in the s's (W[1]–W[25])

Single invariants:

| | | | | |
|---|---|---|---|---|
| W[1] = s12 | W[2] = s23 | W[3] = s34 | W[4] = s45 | W[5] = s15 |

Sums of two adjacent:

| | | | | |
|---|---|---|---|---|
| W[6] = s34 + s45 | W[7] = s15 + s45 | W[8] = s12 + s15 | W[9] = s12 + s23 | W[10] = s23 + s34 |

Differences of two:

| | | | | |
|---|---|---|---|---|
| W[11] = s12 − s45 | W[12] = s23 − s15 | W[13] = s34 − s12 | W[14] = s45 − s23 | W[15] = s15 − s34 |

Linear combos of three:

| | | | | |
|---|---|---|---|---|
| W[16] = s12 + s23 − s45 | W[17] = s23 + s34 − s15 | W[18] = s34 + s45 − s12 | W[19] = s15 + s45 − s23 | W[20] = s12 + s15 − s34 |

Linear combos of four:

| | | | | |
|---|---|---|---|---|
| W[21] = s34 + s45 − s12 − s23 | W[22] = s15 + s45 − s23 − s34 | W[23] = s12 + s15 − s34 − s45 | W[24] = s12 + s23 − s15 − s45 | W[25] = s23 + s34 − s12 − s15 |

### Square-root letters (W[26]–W[31])

Each of W[26]–W[30] has the form `(P − eps5)/(P + eps5)` where `P` is a
bilinear in the `s`'s:

| Letter | P (so that letter = (P − eps5)/(P + eps5)) |
|--------|------|
| W[26] | −s12·s15 + s12·s23 − s23·s34 − s15·s45 + s34·s45 |
| W[27] | −s12·s15 − s12·s23 + s23·s34 + s15·s45 − s34·s45 |
| W[28] | +s12·s15 − s12·s23 − s23·s34 − s15·s45 + s34·s45 |
| W[29] | −s12·s15 + s12·s23 − s23·s34 + s15·s45 − s34·s45 |
| W[30] | +s12·s15 − s12·s23 + s23·s34 − s15·s45 − s34·s45 |

The five `P`'s are the five distinct sign flips (even number of flips each)
of the bilinear `s12·s15 − s12·s23 + s23·s34 + s15·s45 − s34·s45`,
reflecting the cyclic structure of the pentagon.

Finally:

```
W[31] = eps5^2 = Delta5
```

the polynomial under the square root itself. Although `W[31]` is rationally
expressible in the `s`'s, it is kept as a distinct letter because
`d log(eps5^2) = 2 d log(eps5)` ties it to the same square-root branch as
W[26]–W[30].

## Square-root handling

The pentagon alphabet is **not Galois-invariant**: the 5 algebraic letters
`W[26]-W[30]` of the form `(P - eps5)/(P + eps5)` are listed without their
conjugates `(P + eps5)/(P - eps5) = 1/W[i]`. Consequently the integrability
minors contain `1/Sqrt[Delta5]` factors that do **not** cancel.

[SymbolBootstrap.wl](../SymbolBootstrap.wl) handles this automatically:
`GenSqrtD` detects `Sqrt[Delta5]`, Z2-reduces the square-root set,
rationalizes denominators, and `GenIntRelMat` keeps `sq[i]` symbolic in the
minors, extracting rational coefficients via `CoefficientArrays`. All
equations stay in `QQ`, so SparseRREF works natively. No special handling
is required from the caller beyond substituting `eps5 -> Sqrt[Delta5]`
(via `RootDef`) in the alphabet expressions.

## Cyclic symmetry projection matrix

The cyclic transformation `{s12->s23, s23->s34, s34->s45, s45->s15, s15->s12}`
acts on the 5 Mandelstam invariants and induces a permutation on the 31
letters. The action splits the alphabet into 6 five-cycles (W[1–5], W[6–10],
W[11–15], W[16–20], W[21–25], W[26–30]) plus one fixed point (W[31]), forming
the Z5 cyclic group.

The projection matrix `M` encodes the action: `Log[W[i]_new] = Sum_j M[[i,j]]*Log[W[j]]`.
For the cyclic case, `M` is a permutation matrix with one `+1` per row.

```mathematica
$ProjectRoot = "/Users/windfolgen/GitRepos/symbology";
SetDirectory[$ProjectRoot];
Get[FileNameJoin[{$ProjectRoot, "SymmetryProjection.wl"}]];
Get[FileNameJoin[{$ProjectRoot, "data_pentagon", "alphabet.wl"}]];

cyclicRep = {s12 -> s23, s23 -> s34, s34 -> s45, s45 -> s15, s15 -> s12};
projMat = GenerateSymmetryProjection[cyclicRep, LetterRep, RootDef];
Export[FileNameJoin[{$ProjectRoot, "data_pentagon", "cycmat.wxf"}], projMat];
```

| Property | Value |
|----------|-------|
| Dimensions | `{31, 31}` |
| nnz | 31 |
| Group | Z5 (cyclic) |
| Verification | `M^5 == IdentityMatrix[31]` (exact) |

### How `GenerateSymmetryProjection` works

1. For each letter W[i], apply the variable replacement to its expression
2. **Direct match**: check if `exprNew == ±W[j]` (coefficient +1) or `exprNew == ±1/W[j]` (coefficient -1)
3. **`FindLetterLinearRelation` fallback** (for square-root letters that don't match directly): try pairs of existing letters as basis, find integer relation `{c1, c2, c3}` with `c1*Log[b1] + c2*Log[b2] + c3*Log[target] = 0`, giving `Log[target] = -(c1/c3)*Log[b1] - (c2/c3)*Log[b2]`
4. Assemble `result = {Log[W[1]_new], ..., Log[W[31]_new]}` and extract `M = CoefficientArrays[result, Log/@alphabet][[2]]`

## Flip symmetry projection matrix

The flip transformation reverses the order of the external legs
`{1, 2, 3, 4, 5} -> {5, 4, 3, 2, 1}`. On the 5 Mandelstam invariants it acts as
`{s12 -> s15, s23 -> s45, s34 -> s34, s45 -> s23, s15 -> s12}` (s34 and the
diagonal structure are preserved). It is an involution, generating the Z2
dihedral half of the full D5 symmetry of the pentagon.

The induced action on the 31 letters splits the alphabet into **12 two-cycles**
plus **7 fixed points**:

| 2-cycles (swaps) | Fixed points |
|------------------|--------------|
| (W[1], W[5]), (W[2], W[4]), (W[6], W[10]), (W[7], W[9]), (W[11], W[12]), (W[13], W[15]), (W[16], W[19]), (W[17], W[18]), (W[21], W[22]), (W[23], W[25]), (W[26], W[27]), (W[28], W[30]) | W[3], W[8], W[14], W[20], W[24], W[29], W[31] |

The projection matrix `M` encodes the action:
`Log[W[i]_new] = Sum_j M[[i,j]]*Log[W[j]]`. For the flip case, `M` is a
permutation matrix with one `+1` per row (an involution, so `M = M^T = M^-1`).

```mathematica
$ProjectRoot = "/Users/windfolgen/GitRepos/symbology";
SetDirectory[$ProjectRoot];
Get[FileNameJoin[{$ProjectRoot, "SymmetryProjection.wl"}]];
Get[FileNameJoin[{$ProjectRoot, "data_pentagon", "alphabet.wl"}]];

flipRep = {s12 -> s15, s23 -> s45, s34 -> s34, s45 -> s23, s15 -> s12};
projMat = GenerateSymmetryProjection[flipRep, LetterRep, RootDef];
Export[FileNameJoin[{$ProjectRoot, "data_pentagon", "flipmat.wxf"}], projMat];
```

| Property | Value |
|----------|-------|
| Dimensions | `{31, 31}` |
| nnz | 31 |
| Group | Z2 (flip / dihedral half of D5) |
| Cycle structure | 12 transpositions + 7 fixed points |
| Verification | `M^2 == IdentityMatrix[31]` (exact) |
