# 4-Point Form Factor Alphabet (n = 93)

## Reference

Variable definitions follow Eq. (3) of **arXiv:2212.02410**.

## Independent variables

The alphabet is built on **5 independent kinematic variables** derived from
4-point form factor kinematics:

```
u_i = (p_i + p_{i+1})^2 / q^2,      i = {1, 2, 3, 4}   (cyclic: 5 ≡ 1, 6 ≡ 2, ...)
v_i = (p_i + p_{i+1} + p_{i+2})^2 / q^2,   i = {1, 2, 3, 4}
```

Subject to momentum conservation and the on-shell conditions, only three `u`'s
and two `v`'s are independent. The independent set used here is:

```
{u1, u2, u3, v1, v2}
```

with the dependent variables expressed as:

```
u4 = 1 + u2 - v1 - v2
v3 = 1 + u3 - u1 - v2
```

## The 93 letters

The alphabet is defined in [alphabet.wl](alphabet.wl) as `alphabetf = {W[i] -> expr}`
together with `sqrtrep`. The full expressions are obtained by applying `sqrtrep`
to the right-hand sides.

## Symmetry transformations

Each transformation is a substitution rule on the independent variables (and/or
square roots). It induces a linear map `M` on the letter logarithms:
`Log[W[i]_new] = Sum_j M[[i,j]] * Log[W[j]]`. The matrix `M` is generated by
`GetLetterTransformationTensor["4pFF", transName]` in
[SymbolBootstrap.wl](../SymbolBootstrap.wl).

### Cyclic transformation (Z4)

Cyclic shift of the external legs `{1, 2, 3, 4} -> {2, 3, 4, 1}`, which on the
independent variables acts as:

```
{u1 -> u2,  u2 -> u3,  u3 -> u4 = 1 + u2 - v1 - v2,
 v1 -> v2,  v2 -> v3 = 1 + u3 - u1 - v2}
```

- Output file: `cycmat.wxf`
- Group: Z4 (cyclic)
- Verification: `M^4 == IdentityMatrix[93]`

### Flip transformation (Z2)

Reversal of the external leg ordering `{1, 2, 3, 4} -> {4, 3, 2, 1}`, which on
the independent variables acts as:

```
{u1 -> u4 = 1 + u2 - v1 - v2,  u2 -> u3,  u3 -> u2,
 v1 -> v3 = 1 + u3 - u1 - v2,  v2 -> v2}
```

- Output file: `flipmat.wxf`
- Group: Z2 (flip / dihedral half of D4)
- Verification: `M^2 == IdentityMatrix[93]`

### Galois transformations (Z2 each)

Each Galois conjugation flips the sign of one independent square root while
leaving all rational variables and the other roots untouched. There are 5
independent Galois transformations, one per square root:

| Name | Substitution rule | Letters affected | Output file |
|------|-------------------|------------------|-------------|
| Galois1a | `sqrt1a -> -sqrt1a` | W[59], W[60], W[63], W[64], W[91] (5 letters) | `galois1a.wxf` |
| Galois1b | `sqrt1b -> -sqrt1b` | W[57], W[58], W[61], W[62], W[90] (5 letters) | `galois1b.wxf` |
| Galois2a | `sqrt2a -> -sqrt2a` | W[65]–W[68], W[92] (5 letters) | `galois2a.wxf` |
| Galois2b | `sqrt2b -> -sqrt2b` | W[69]–W[72], W[85] (5 letters) | `galois2b.wxf` |
| Galois3  | `sqrt3 -> -sqrt3`   | W[74]–W[93] (20 letters) | `galois3.wxf` |

Each Galois transformation is an involution: `M^2 == IdentityMatrix[93]`.

## Generation

```mathematica
$ProjectRoot = "/Users/windfolgen/GitRepos/symbology";
SetDirectory[$ProjectRoot];
Get[FileNameJoin[{$ProjectRoot, "SymbolBootstrap.wl"}]];
Get[FileNameJoin[{$ProjectRoot, "data_4pformfactor", "alphabet.wl"}]];

DeclareAlphabet["4pFF", Table[W[i], {i, 1, 93}]];
SetAlphabetExpression["4pFF", alphabetf[[All, 2]] /. sqrtrep];

(* dlogmat (integrability) *)
dlogmat = GetIntegrabilityTensor["4pFF"];
Export[FileNameJoin[{$ProjectRoot, "data_4pformfactor", "dlogmat_4pformfactor.wxf"}], dlogmat];

(* Symmetry transformation matrices *)
cyclicMap = {u1 -> u2, u2 -> u3, u3 -> 1 + u2 - v1 - v2, v1 -> v2, v2 -> 1 + u3 - u1 - v2};
flipMap   = {u1 -> 1 + u2 - v1 - v2, u2 -> u3, u3 -> u2, v1 -> 1 + u3 - u1 - v2, v2 -> v2};

SetLetterTransformation["4pFF", "Cyclic", cyclicMap];
cycmat = GetLetterTransformationTensor["4pFF", "Cyclic"];
Export[FileNameJoin[{$ProjectRoot, "data_4pformfactor", "cycmat.wxf"}], cycmat];

SetLetterTransformation["4pFF", "Flip", flipMap];
flipmat = GetLetterTransformationTensor["4pFF", "Flip"];
Export[FileNameJoin[{$ProjectRoot, "data_4pformfactor", "flipmat.wxf"}], flipmat];

(* Galois conjugations: flip sign of each sqrt independently *)
Do[
  With[{sym = sqrtrep[[k, 1]], rule = {sqrtrep[[k, 2]] -> -sqrtrep[[k, 2]]}, name = {"Galois1a", "Galois1b", "Galois2a", "Galois2b", "Galois3"}[[k]]},
    SetLetterTransformation["4pFF", name, rule];
    mat = GetLetterTransformationTensor["4pFF", name];
    Export[FileNameJoin[{$ProjectRoot, "data_4pformfactor", ToLowerCase[name] <> ".wxf"}], mat];
  ],
  {k, Length[sqrtrep]}
];
```
